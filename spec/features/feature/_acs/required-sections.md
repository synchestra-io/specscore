# AC: required-sections

**Requirement:** feature#req:required-sections

Every feature README contains these sections: Summary, Problem, Behavior, Acceptance Criteria, and Outstanding Questions. A README missing any required section is rejected by validation.

**See also:** [outstanding-questions](outstanding-questions.md) (empty-state rules for Outstanding Questions), [ac-section](ac-section.md) (empty-state rules for Acceptance Criteria), [title-format](title-format.md) (title format rule), [status-field](status-field.md) (status field rule)
