# AC: slug-format

**Requirement:** feature#req:slug-format

Feature directory names use only lowercase letters, numbers, and hyphens. Names containing underscores, spaces, uppercase letters, or special characters are rejected by validation.

**See also:** [directory-readme](directory-readme.md) (the directory must also contain a README), [path-identification](path-identification.md) (slugs compose into feature identifiers)
