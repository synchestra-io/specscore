# AC: ac-section

**Requirement:** feature#req:ac-section

The Acceptance Criteria section is always present. When no ACs are defined, it contains "Not defined yet." and the Outstanding Questions section includes "Acceptance criteria not yet defined for this feature."

**See also:** [outstanding-questions](outstanding-questions.md) (the Outstanding Question this AC triggers), [required-sections](required-sections.md) (AC is a required section)
