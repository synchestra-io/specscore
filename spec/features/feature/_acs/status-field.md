# AC: status-field

**Requirement:** feature#req:status-field

A `**Status:**` field appears immediately after the title and contains one of the four valid values: `Conceptual`, `In Progress`, `Stable`, or `Deprecated`. A missing status field or an unrecognized value is rejected by validation.

**See also:** [title-format](title-format.md) (status follows the title), [required-sections](required-sections.md) (status is a required structural element)
