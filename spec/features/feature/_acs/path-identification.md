# AC: path-identification

**Requirement:** feature#req:path-identification

Features are identified by their path relative to `spec/features/`. This path-based identifier is the canonical reference used in development plans, source references, and spec tooling.

**See also:** [slug-format](slug-format.md) (each path segment is a valid slug), [index-completeness](index-completeness.md) (top-level identifiers must appear in the index)
