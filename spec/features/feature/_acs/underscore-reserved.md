# AC: underscore-reserved

**Requirement:** feature#req:underscore-reserved

Directories prefixed with `_` are not treated as sub-features. They do not appear in the feature index or the Contents table of a parent feature.

**See also:** [contents-when-children](contents-when-children.md) (Contents table excludes `_` dirs), [index-completeness](index-completeness.md) (feature index excludes `_` dirs)
