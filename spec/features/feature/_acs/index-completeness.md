# AC: index-completeness

**Requirement:** feature#req:index-completeness

The feature index (`spec/features/README.md`) lists every top-level feature. A feature directory that exists under `spec/features/` but is not listed in the index is a validation error.

**See also:** [underscore-reserved](underscore-reserved.md) (`_`-prefixed directories are excluded from the index), [path-identification](path-identification.md) (the index uses path-based identifiers)
