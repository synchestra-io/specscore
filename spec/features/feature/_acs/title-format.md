# AC: title-format

**Requirement:** feature#req:title-format

The first line of a feature README is a level-1 heading with the `Feature:` prefix. A README whose title does not match `# Feature: {Title}` is rejected by validation.

**See also:** [status-field](status-field.md) (status appears immediately after the title), [required-sections](required-sections.md) (title is one of the required structural elements)
